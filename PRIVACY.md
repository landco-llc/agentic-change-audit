# Privacy

This policy applies to the **Agentic Change Audit Codex Plugin** distributed from this repository.

English is the canonical version. The Japanese and Traditional Chinese sections below are provided for convenience and are materially equivalent.

Public URL after merge:

## What this Plugin is

This is a **skills-only Plugin**. It consists of a manifest and a bundled Skill: instructions, standards, and templates in text form.

It includes **no MCP server**, no ChatGPT app, no connector, no external service, no telemetry, no analytics, no authentication flow, and no network client.

## Data collection

**The Plugin itself does not collect, transmit, sell, or share user data.**

There is no telemetry and no analytics in this Plugin. Nothing in it reports usage, events, prompts, or results anywhere.

L&Co.LLC does not receive your task contents merely because the Plugin is installed.

## What the Plugin reads

The Plugin reads only the data that is already made available to the active ChatGPT or Codex task, under your own environment and your own permissions. It declares exactly one capability: `Read`.

Any repository, file, or tool data the host product accesses on your behalf remains governed by the host product and by the tools you have configured. This policy does not change or override those terms.

## Your inputs

Do not paste secrets, credentials, tokens, or personal data unnecessarily into an audit task. The Plugin cannot remove information you have already supplied to the host product.

## Audit outputs

An audit result may include repository paths, commit SHAs, branch names, filenames, quoted evidence, and findings, because an evidence-first audit records what it inspected.

You control where those outputs are stored, pasted, or shared. Review a result before sharing it outside your organization.

## Scope of this policy

This policy applies only to this skills-only Plugin at its current version.

If a future version adds an MCP server, an app, a connector, telemetry, analytics, an authentication flow, or any hosted service, that version requires a new privacy policy and a new review. It is not covered by this document.

### Pluginが読み取る範囲
